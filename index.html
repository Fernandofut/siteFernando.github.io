Bem-vindo ao site

Este site foi criado para demonstrar a estrutura de uma página HTML utilizando apenas HTML, sem CSS.

Aqui você pode navegar pelo menu da esquerda para acessar outras páginas do site.

O objetivo deste projeto é aprender a organizar um site com cabeçalho, menus laterais, conteúdo central e rodapé.
